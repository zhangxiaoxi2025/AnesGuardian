## 🎨 功能演示

### 评估流程

```
1. 选择患者 → 2. 开始评估 → 3. 实时监控 → 4. 查看报告
```

### 药物分析流程

```
1. 输入药物 → 2. 检测相互作用 → 3. 查看详情 → 4. 获取建议
```

### 医疗记录流程

```
1. 上传图片 → 2. AI 分析 → 3. 医生审核 → 4. 保存记录
```
